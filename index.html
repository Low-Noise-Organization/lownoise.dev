01 — Sobre Mi

El sistema detrás
del código

Con 19 años y nacido en Córdoba, España, no espero a que alguien me enseñe: busco, fallo, entiendo y construyo. Mi entorno principal es Fedora Linux, mi herramienta de depuración favorita es el terminal y mis proyectos no terminan hasta que el código es lo que yo busco.

Me especializo en el backend: la lógica que no se ve pero que sostiene todo. Java y Kotlin para aplicaciones robustas; Python para automatización y scripting, Bash para cuando necesito que el sistema haga lo que yo quiero, no al revés.

Cada proyecto que lanzo es un experimento: ¿qué puedo aprender construyendo esto? El resultado importa, pero el proceso de entenderlo todo importa más.
